# LISP

## An Introduction to LISP Using the compilers conforming to Common Lisp of: 
## Steel Bank Common Lisp ~ SBCL & CLISP

* lisp_presentation.odp

* demo_radio.lisp
* demo_2.lisp
* demo_math.lisp

* demo_date.lisp
* demo_file.lisp
